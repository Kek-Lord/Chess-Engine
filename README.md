## Chess Engine Overview ♟
This chess engine is implemented using JavaFX and can be run using the maven sidebar in IntelliJ -> ChessEngine -> Plugins -> javafx -> javafx:run.

### Current Features
- Chess board GUI
- Click and move pieces
- Basic Evaluation bar
- Play against a random move chess bot
- Choose to play as White, Black, or Random
- Flip the board to see from other side
- Watch AI play itself

## Features Roadmap
- FEN string imports
- More accurate evaluation bar
- Move evaluations for the bot
- Playing against stockfish
- Alpha-Beta pruning
- Board and Piece appearance customisation
- API to allow users to build bots for this framework
